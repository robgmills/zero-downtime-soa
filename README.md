# zero-downtime-soa

## Quick Start
### Step 1 - Get the dependencies
Simply execute the `bootstrap.sh` script to download and install all the
necessary dependencies including:

- kafka 2.10-0.8.2.0

Dependencies are downloaded to an `opt/` directory in this project root.
